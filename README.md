# Vehicle-Rental-System
The Vehicle Rental Management System is a Java-based desktop application designed to automate and simplify the operations of vehicle rental services. The system allows users to register, log in, view available vehicles, and book vehicles for specific rental periods, while administrators can manage vehicle records, monitor bookings, and maintain overall system operations. By replacing manual rental management methods with a digital platform, the system improves efficiency, reduces errors, and provides a smoother user experience.

The application is developed using Java Swing for the graphical user interface and MySQL for the backend database, with JDBC used for database connectivity. It follows object-oriented programming principles and a modular structure, which helps in maintaining and expanding the system easily. The system includes several modules such as the Admin Module for managing vehicles and bookings, the Vehicle Management Module for adding and maintaining vehicle records, the Booking Module for handling vehicle reservations and simulated payments, and the User Module for registration, login, and viewing booking history.

Overall, the project demonstrates the practical implementation of Java GUI development, database integration, and modular software design. It provides a reliable and user-friendly platform for managing vehicle rental operations and can be further enhanced in the future by integrating features such as real-time payment gateways, mobile application support, and advanced analytics for administrators.
